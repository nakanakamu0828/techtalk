# Sorcery
#ruby #RubyOnRails
「Sorcery」はRuby On Railsでユーザー認証機能を提供するgemです。  
ユーザー認証の代表格「devise」と異なり、認証に必要なサービスロジックのみを「Sorcery」は提供しています。
