# npm

>npmの正式名称は、Node Package Managerです。
>npmの正式名称からも解るように、Node.jsのパッケージ（Package ）を管理する（Manager）ツールです。
>Node.jsのパッケージ（Package）とは、予め用意された便利な機能をまとめたものです。
