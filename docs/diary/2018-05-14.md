# 2018/05/14
## nuxt-edgeの導入方法
#nuxtjs


以下のように`nuxt`を削除して`nuxt-edge`をインストール
```
$ npm uninstall nuxt @nuxtjs/axios --save
$ npm install nuxt-edge @nuxtjs/axios -save
```

あとは、利用している他のライブラリなど依存やエラーを回避する
