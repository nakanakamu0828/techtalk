## DockerでMySQLコンテナにアクセスする方法
#infra, #Docker

Dockerプロセスを確認してから、bash接続する。
繋がった後は通常通り`mysql`コマンドで接続。

```command
$ docker ps
$ docker exec -it [コンテナID] /bin/bash

$ mysql -uroot -p
```
